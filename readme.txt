== Description ==

SearchMindAI Enhanced Search replaces the default WordPress and WooCommerce search functionality with an AI-powered search engine that delivers improved and more relevant search results based on the content of your website. This ensures users can easily find products, posts, and pages through smarter, context-aware search capabilities.

Harness the power of OpenAI with SearchMindAI to revolutionize your store's search experience. Install our plugin today and experience the difference it can make for your business.


Key features:
* Replaces default WordPress search results with AI-enhanced results.
* Provides more accurate results by analyzing the content of your website.
* Optimized for WooCommerce to help customers find products more effectively.
* Easy to set up by registering with SearchMindAI to complete the configuration and obtain your credentials.

== Easy Installation and Configuration ==

This version of SearchMindAI’s plugin is **FREE** and is designed for easy installation and configuration. With a user-friendly interface, you can customize settings to best suit your store’s needs. Comprehensive documentation and support are provided to ensure a smooth integration process.

== Installation ==

1. Upload the `searchmindai-enhanced-search` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Update your permalinks by going to Settings > Permalinks and saving the changes.
4. Register on [SearchMindAI.com](https://www.searchmindai.com/registration) to complete the configuration of your store and get your credentials.
5. After registration, go to the WordPress admin page > Settings > SearchMindAI settings, and copy the provided URLs.
6. Paste the copied URLs into the SearchMindAI panel to complete your store configuration.

== Frequently Asked Questions ==

= How does the plugin improve search results? =
The plugin uses AI to analyze and understand the content on your website, delivering results that are more contextually relevant to the user's search queries.

= Is it compatible with WooCommerce? =
Yes! It is fully compatible with WooCommerce and enhances the search functionality for both products and regular WordPress content.

= Does the plugin connect to any external services? =
Yes, this plugin connects to the SearchMindAI API in order to perform advanced search queries. The API processes the search queries and returns relevant results based on the product catalog.

= What data is sent to the API? =
The plugin sends search terms entered by users, pagination data (if applicable), and other parameters like the API key and document type for searching products or pages. No personal user data is sent to the API.

= What is the external service used? =
- **Service Name**: SearchMindAI API
- **Service URL**: [https://api.searchmindai.com/](https://api.searchmindai.com/)
- **Service Privacy Policy**: [https://www.searchmindai.com/privacy-policy](https://www.searchmindai.com/privacy-policy)

= Why is the external service required? =
The SearchMindAI API is used to process search queries and provide advanced search capabilities that are not available by default in WooCommerce or WordPress.

= Can I use the plugin without connecting to the external service? =
No, the plugin’s core functionality relies on the SearchMindAI API. Without connecting to the external service, the plugin will not function as intended.
